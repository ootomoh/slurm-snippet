# Neosnippet Slurm jobscript extension
## How to install
1. Find path of `sh.snip`.  
   (`~/.cache/dein/repos/github.com/Shougo/neosnippet-snippets/neosnippets/sh.snip`, ...)
2. Append `sh-extension.snip` to `sh.snip`

## How to use
1. `vim jobscript.sh`
2. Write 'slurm'
3. Expand snippet (Ctrl-k)
